# 🚀 馬上使用 - AI Prompt 優化工具

> 3 分鐘快速上手指南

## 📋 前置需求

- Python 3.8 或更高版本
- 一個 LLM API Key (推薦使用 Google Gemini)

## ⚡ 快速開始 (推薦: Gemini)

### 1️⃣ 安裝依賴

```bash
pip install -r requirements.txt
```

### 2️⃣ 設置 API Key

**方法 A: 使用介面輸入 (最簡單)**
```bash
streamlit run app.py
```
啟動後,在側邊欄的「Gemini API Key」欄位直接輸入你的 API Key,點擊「確認」即可。

**方法 B: 使用環境變數**
```bash
# 創建 .env 文件
echo "GEMINI_API_KEY=你的API密鑰" > .env

# 啟動應用
streamlit run app.py
```

### 3️⃣ 開始使用

1. 在瀏覽器打開 `http://localhost:8501`
2. 在輸入框貼上你想優化的提示詞
3. 點擊「分析提示」
4. 回答改進問題
5. 點擊「生成優化提示」
6. 獲得專業優化後的提示詞! 🎉

## 🔑 如何獲取 Gemini API Key?

1. 前往 [Google AI Studio](https://aistudio.google.com/app/apikey)
2. 登入 Google 帳號
3. 點擊「Get API Key」→「Create API Key」
4. 複製生成的 API Key

**免費額度**: Gemini API 提供免費配額,足夠日常使用!

## 🆘 常見問題

### Q: 為什麼推薦 Gemini?
A: Gemini API 免費、快速、易於申請,非常適合個人使用。

### Q: 可以離線使用嗎?
A: 不行,需要連接 LLM API 服務。

### Q: 支援 GPT-4 嗎?
A: 目前 OpenAI 整合尚未完成,請使用 Gemini 或 Claude。

### Q: 分析一次要多久?
A: 使用 Gemini Flash 模型約 3-5 秒。

### Q: API Key 安全嗎?
A:
- 在介面輸入的 API Key 只存在記憶體中,關閉瀏覽器即清除
- 使用 .env 文件請確保不要提交到 Git
- .env 已預設在 .gitignore 中
